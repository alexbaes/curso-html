Preguntas sobre HTML

Preguntas básicas

Qué es HTML y qué papel desempeña en el desarrollo web?

HTML significa "HyperText Markup Language" y es el lenguaje de marcado que se utiliza para crear páginas web. Define la estructura y el contenido de una página web y se utiliza junto con otros lenguajes como CSS y JavaScript para darle estilo y funcionalidad a la página.

¿Cuáles son las diferentes versiones de HTML?

Las diferentes versiones oficiales de HTML son HTML 2, HTML 3.2, HML 4.4, HTML 4.01 Y HTML 5. XHTML es una forma más avanzada pensada para sustituir HTML pero se quedó integrado en HTML 5.

¿Qué es una etiqueta en HTML y cómo se utiliza?

Una etiqueta en HTML es una símbolo que le indica al navegador cómo interpretar y mostrar el contenido de una página web. Las etiquetas están encerradas entre corchetes angulares (< >) y pueden tener atributos que modifican su comportamiento.

¿Qué es el atributo "alt" en una etiqueta de imagen y por qué es importante?

El atributo "alt" en una etiqueta de imagen describe brevemente el contenido de la imagen en caso de que no se pueda mostrar correctamente. Es importante para la accesibilidad web y para los motores de búsqueda, que utilizan esta información para indexar el contenido de la página.

¿Cuál es la diferencia entre una lista ordenada y una lista desordenada en HTML?

Una lista ordenada en HTML se crea utilizando la etiqueta < ol > y muestra los elementos de la lista en orden numérico o alfabético. Una lista desordenada se crea utilizando la etiqueta < ul > y muestra los elementos de la lista sin un orden específico.

¿Qué son las tablas en HTML y cuál es su uso principal?

Las tablas en HTML se utilizan para organizar datos en filas y columnas. Se crean utilizando la etiqueta < table > y pueden tener encabezados, pies de tabla y celdas con diferentes formatos y estilos.

¿Cómo se crea un enlace en HTML y qué atributos puede tener?

Un enlace en HTML se crea utilizando la etiqueta y se utiliza para navegar por el sitio web o vincular una página web externa. Los atributos más comunes son 'href', 'target', 'rel', 'download', 'title', 'type', aunque hay algunos más.

¿Qué es un formulario en HTML y para qué se utiliza?

. Un formulario en HTML se utiliza para recopilar información de los usuarios, como nombres, direcciones de correo electrónico y contraseñas. Los formularios se crean utilizando la etiqueta < form > y pueden contener diferentes tipos de elementos, como campos de texto, botones, selectores...

¿Qué es el atributo "id" en HTML y cuál es su importancia?

El atributo "id" en HTML se utiliza para identificar de manera única un elemento en una página web. Se utiliza para enlazar elementos específicos a través de anclas y para aplicar estilos de CSS específicos a elementos individuales.

¿Qué es el modelo de caja en CSS y cómo se relaciona con el HTML?

El modelo de caja en CSS es la forma en que se estructura el contenido de una página web. Cada elemento HTML se trata como una caja rectangular que tiene un contenido, un relleno, un borde y un margen

Preguntas avanzadas

¿Cuál es la diferencia entre un elemento "inline" y un elemento "block"? Proporcione ejemplos.

Los elementos "inline" son aquellos que ocupan solo el ancho necesario en línea, mientras que los elementos "block" ocupan todo el ancho disponible y siempre comienzan en una nueva línea. Por ejemplo, un elemento "inline" podría ser una etiqueta "span" mientras que un elemento "block" podría ser una etiqueta "div".

¿Qué son los elementos "semanticos" en HTML y por qué son importantes?

Los elementos "semanticos" en HTML son aquellos que describen el contenido de la página de manera significativa, en lugar de simplemente presentarlo visualmente. Son importantes porque ayudan a mejorar la accesibilidad y la estructura de la página, lo que a su vez beneficia a la optimización para motores de búsqueda.

¿Cuál es la diferencia entre "HTML" y "XHTML"?

La principal diferencia entre HTML y XHTML es que XHTML es una versión más estricta y basada en XML de HTML. XHTML tiene reglas más estrictas para el marcado y debe ser escrito correctamente para que un navegador pueda interpretarlo correctamente. Además, XHTML no admite algunas etiquetas y atributos de HTML más antiguos.

¿Qué es "WAI-ARIA" y cómo se utiliza en HTML?

"WAI-ARIA" significa "Web Accessibility Initiative - Accessible Rich Internet Applications". Se utiliza para mejorar la accesibilidad de las aplicaciones web, proporcionando información adicional sobre la funcionalidad y el comportamiento de los elementos de la página para ayudar a los usuarios con discapacidades a comprender la página.

¿Cuál es la diferencia entre una "cookie" y una "sesión" en HTML?

Una "cookie" es un pequeño archivo que se almacena en el navegador del usuario, mientras que una "sesión" es un conjunto de variables de servidor que se almacenan mientras un usuario interactúa con una página web. Las cookies se utilizan generalmente para recordar la información del usuario, como las preferencias, mientras que las sesiones se utilizan para almacenar información temporal, como los elementos que se han agregado al carrito de compras.

¿Cómo se manejan los caracteres especiales en HTML?

Los caracteres especiales en HTML se manejan utilizando entidades HTML, como "<" para "<" o "&" para "&". Estas entidades se utilizan para representar caracteres que no se pueden escribir directamente en el código HTML.

¿Cuál es la diferencia entre las etiquetas "div" y "span"? Proporcione ejemplos de uso.

La principal diferencia entre estas dos etiquetas es que < div > es una etiqueta de bloque mientras que < span > es de línea. Las primeras se utilizan para crear secciones grandes de contenido, y la segunda para añadir estilos a textos.

¿Qué son los "Web Components" en HTML y cómo se crean?

Los "Web Components" son un conjunto de tecnologías que permiten la creación de elementos personalizados reutilizables para su uso en múltiples páginas web. Se crean utilizando HTML, CSS y JavaScript y se pueden utilizar en cualquier sitio web. Los componentes web están hechos de tres partes principales: la plantilla, el script y el estilo.

¿Qué es "Canvas" en HTML y para qué se utiliza?

"Canvas" es un elemento HTML que se utiliza para dibujar gráficos, animaciones y otros elementos visuales en una página web utilizando JavaScript. Es similar a una hoja de papel en la que se puede dibujar cualquier cosa utilizando herramientas de dibujo. Es comúnmente utilizado para juegos y animaciones en línea..

¿Qué son los "Data Attributes" en HTML y cómo se utilizan?

Los "Data Attributes" en HTML son atributos personalizados que se pueden agregar a los elementos HTML para almacenar información adicional en el propio elemento. Se utilizan para asociar datos con un elemento sin afectar a la apariencia ni al comportamiento del mismo. Estos atributos se definen con el prefijo "data-" y se pueden acceder a ellos a través del objeto "dataset" en JavaScript.

¿Cómo se almacenan los datos en HTML5?

HTML5 en sí mismo no almacena datos, pero proporciona varias API que permiten el almacenamiento de datos en el lado del cliente. Estas API incluyen localStorage, sessionStorage, IndexedDB y WebSQL.

  • localStorage y sessionStorage permiten almacenar datos clave-valor en el navegador del usuario, con la diferencia de que los datos almacenados en sessionStorage se eliminan cuando se cierra la pestaña del navegador, mientras que los datos almacenados en localStorage se conservan incluso después de cerrar el navegador.
  • IndexedDB es una base de datos de objetos transaccional que permite el almacenamiento de grandes cantidades de datos estructurados en el navegador.
  • WebSQL es otra API de base de datos relacional que permite el almacenamiento de datos estructurados en el lado del cliente, pero no está disponible en todos los navegadores y ha sido desaprobado en favor de IndexedDB.
